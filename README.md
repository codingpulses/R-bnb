<h2>🛠️ Installation Steps:</h2>

<p>1. Move to each folder.</p>

```
cd api/cd client
```

<p>2. In api run the command.</p>

```
node index.js
```

<p>3. In client run the command.</p>

```
npm run dev
```

<p>4. Go on the</p>

```
 localhost:5173
```
